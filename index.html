Live demos

The following projects have live demos hosted on GitHub:

  • Mazedoku: a generator for a Sudoku variant.
  • Duokoban: a Sokoban variant that is played by two players on a single keyboard! (This is a static archive of the version that was originally hosted on Heroku, which supported rating levels and submitting new ones.)
  • Maze generator: generates truly uniform mazes using Wilson's algorithm, on a rectangular or hexagonal grid.
  • Setko Puzzler 67: a peg-jumping puzzle, including a solver that can reveal optimal moves.
  • Hexagon puzzle: a score optimization puzzle on a hexagon. The source code repository also contains a solver to find optimal solutions.
